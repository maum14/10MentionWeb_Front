D- Alignement

Les éléments sont organisés soit horizontalement (par défaut), soit verticalement.

  • si vos éléments sont organisés horizontalement : l'axe principale (main axis) est l'axe horizontale, l'axe secondaire (cross axis) est l'axe vertical.
  • si vos éléments sont organisés verticalement : l'axe principale (main axis) est l'axe verticale, l'axe secondaire (cross axis) est l'axe horizontale.

1 - alignement sur l'axe principal

afin d'aligner nos éléments sur l'axe principal: on utilise la propriété du flex justify-content

Cette propriété gère la distribution des emplacements des flex-items suivant la place disponible dans le flex-container. elle le fait , selon le positionnement ede l'axe principal

  • flex-start: aligner au début ( par défaut )
  • flex-end : aligner a la fin
  • center : aligner au centre
  • space between : les éléments sont étirés sur tout l'axe (il y a de l'espace entre eux )
  • space-around: les élément sont étirés sur tout l'axe mais ils laissent aussi de l'espace sur des extrémités
  • space-evenly : les élément son répartis uniformément dans le conteneur d'alignement le long d el'axe principal

2- Alignement sur l'axe secondaire

afin d'aligner nos éléments sur bl'axe secondaire: on utilise la propriété du flex align-items

Elle peit prendre comme valeur:

  • flex-start: aligner au début
  • flex-end: aligner à la fin
  • center aligner au centre
  • stretch: les éléments sont étirés sur tout l'axe (valeur par défaut)
  • baseline: aligne sur la ligne de base
