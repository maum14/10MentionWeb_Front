c- direction

flexbox nous premet d'agencer ces éléments dans le sens que l'on veut. Avec flex-direction,on peut les possitionner verticalement ou encore les inverser. il peut prendre les valeurs suivantes:
-> row : organisés sur une ligne (par défaut)
-> column : organisés sur une colonne
-> row-reverse : organisés sur une ligne mais en ordre inversé
-> column-reverse : organisé sur une colonne , mais en ordre inversé
